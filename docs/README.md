# FYP-ClassNet Documentation

This directory contains formal specification documents for the FYP-ClassNet project.

## Documents

### 1. FYP-ClassNet_SRS.docx
**Software Requirements Specification (SRS)**

This document provides a complete description of all functional and non-functional requirements for the FYP-ClassNet system. It includes:
- System overview and scope
- Detailed functional requirements for all features
- Non-functional requirements (performance, security, usability)
- User classes and operating environment
- External interface requirements
- Database schema requirements

**Target Audience:** Developers, testers, project managers, stakeholders

### 2. FYP-ClassNet_SDD.docx
**Software Design Document (SDD)**

This document describes the architecture and design of the FYP-ClassNet system. It includes:
- System architecture (three-tier design)
- Component design for frontend (React) and backend (Node.js)
- Database schema and data flow diagrams
- User interface design specifications
- Deployment architecture using Docker
- API reference and Socket.io event specifications
- Security and performance design

**Target Audience:** Software developers, system architects, QA teams, maintenance teams

## Technology Stack

The documents describe a system built with:
- **Frontend:** React 18.x
- **Backend:** Node.js with Express.js
- **Database:** SQLite 3.x
- **Real-time Communication:** Socket.io, WebRTC
- **Deployment:** Docker & Docker Compose
